Install Widget

Copy the following script and insert it into your website's HTML source code between the <body> tags. This code must be inserted into every page where you want to display the web widget.

                  
    
                    
  1. <script>
  2. var script = document.createElement("script");
  3. script.type = "text/javascript";
  4. script.src = "https://SUBDOMAIN.ngdesk.com/widgets/chat/CHAT_CHANNEL_ID/chat_widget.js";
  5. document.getElementsByTagName("head")[0].appendChild(script);
  6. </script>
